공통세션 Django 기초 2주차 과제

2주차 세션 정리
1. Django MTV 패턴
  • Model: 데이터의 구조와 DB를 관리
  • Template: 사용자에게 보여지는 화면(HTML)
  • View: Model과 Template 사이에서 로직을 연결 및 제어
2. 페이지 생성
  1. HTML 작성
  2. View 작성
  3. URL 연결
  • 요약: URL로 접속 -> View가 로직 수행 -> 완성된 Template을 보여줌
3. Template 언어
    {% for TL in TLS %}
  • {{ TL|safe }}
    • {% endfor %}
4. 중복되는 html 파일 정리
  • Template 상속: base.html이라는 큰 틀을 만들고, 다른 페이지는 {{"{%"}} extends %}로 이 틀을 그대로 가져다 씀
  • Navbar 분리: Navbar를 _navbar.html로 따로 빼두고, {{"{%"}} include %}로 필요한 곳에 삽입
5. 정적 파일 분리
  • 설정: project 폴더 내에 static 폴더를 생성하고 CSS나 이미지를 저장
  • 불러오기: HTML 상단에 {% load static %}을 먼저 선언
  • CSS 연결: <link rel="stylesheet" href="{% static 'css/project.css' %}">
  • 이미지 연결: <img src="{% static 'images/photo.png' %}" alt="이미지">
